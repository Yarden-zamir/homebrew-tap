## How do I install these formulae?

`brew install yarden-zamir/tap/<formula>`

Or `brew tap yarden-zamir/tap` and then `brew install <formula>`.

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
